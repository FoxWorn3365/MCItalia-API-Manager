# Minecraft Italia API Manager
Un semplice programma rigorosamente in **PHP** per recuperare le informazioni di un server tramite le API di **Minecraft Italia**

## Installazione
Per installare il file sarà sufficente caricare il loader.php (che potrai trovare in **src**) ed includerlo correttamente in una pagina PHP<br>(come mostrato nella index_example.php)
<br>
In caso si possieda **Linux** basterà eseguire il file `install.sh` che puoi torvare in **src** che lancerà un `wget`

## Configurazione
Per far funzionare il programma basterà inserire nel file `conf.key` lo slug del server minecraft desiderato
